Week 11: JSON

In the table below is a list of a few Marvel superheroes along with their super powers. Use JSON to populate the superheroes object in the <script> element at the bottom of this file with the data from the table. Make your object's structure follow the data structure of the table.

Guidelines

  • Each superhero should be represented by their own object within the superheroes "parent" object.
  • Each super power should be a separate string within an array of super powers assigned to their respective superhero. E.g. ["Flight", "Super Strength"]

Loop through one of the arrays you created in the superheroes object and log each super power to the console.
