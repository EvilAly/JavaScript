Week 13: The DOM, Continued

This week we've learned a lot more about DOM manipulation. In this week's assignment, we'll put into practice a few of the new things we've learned.

Follow the steps of parts 1 and 2 to complete this assignment.

Part 1:

  1. Select the .target element below and save it as a variable.
  2. Clone this element, assigning the new clone to a variable
  3. Use classList to add the class cloned to the cloned element (but not the original element!)
  4. Select the .part-1 element below and save it to a variable
  5. Append the cloned element 7 times to your saved .part-1 element (making it a sibling of the original .target div). Do not just write 7 copies of your code - use a loop.
    When you're done, there should be a 4-column grid of 8 circles - one gray and seven green.

HINT: You'll need to clone the original .target element and set its class inside the loop, not before. If you try to clone an element once and repeatedly add it to the DOM, it won't work! You need a new clone for each addition.

Part 2:

  1. Create a new <span> element and assign it to a variable
  2. Use classList to add the class message and the ID part-2-done to it
  3. Set its text to read, "All done"
  4. Select the .part-2 element below and save it to a variable
  5. Append your new element to the .part-2 element

HINT: If the text you added isn't bold and green, something's wrong!
